# Flower Shop (Frontend Practice)

A clean, responsive flower shop landing page built with HTML, CSS, and vanilla JavaScript.

This project was created as a frontend practice project to demonstrate layout structure and styling.

## Sections
- Gallery
- About
- Contact

## Features
- Responsive layout
- Custom CSS styling
- Anchor navigation (single-page sections)
- Clean HTML structure

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## Purpose
This project is part of my frontend portfolio and was built to practice:
- Landing page layout
- Flexbox / responsive styling
- UI structure and spacing
- Basic DOM interactions (optional practice file)

## Notes
This repo may include a separate JavaScript/CSS practice file used for learning and experiments (not part of the main page).
